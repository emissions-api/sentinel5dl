Security Policy¶

Supported Versions¶

We support only the latest version of this library. It has a very simple API and updating to a new version should not be problematic.

Reporting a Vulnerability¶

If you find a security vulnerability, please report it by sending a mail to security@emissions-api.org. We will discuss the problem internally and, if necessary, release a patched version as soon as possible.
